Site Generator Plugin

Welcome to the Site Generator plugin! This plugin allows you to generate temporary web pages based on provided HTML content. You can specify the content for the HTML body in the request body. The 'html' request field accepts a string.

How to Use the Endpoint

To create a new web page, make a POST request to the /page endpoint with the following JSON structure in the request body: 

{
  "html": "<!DOCTYPE html><html lang=\\"en\\"><head><meta charset=\\"UTF-8\\"><title>My Page</title></head><body><h1>Hello, World!</h1><p>This is a simple web page.</p></body></html>"
}

The response will contain the URL of the generated web page. The page will be available for a limited time before it is automatically deleted.

Basic Example

Here's an example of how to create a simple web page with a heading and a paragraph: 

{
  "html": "<!DOCTYPE html><html lang=\\"en\\"><head><meta charset=\\"UTF-8\\"><title>My Page</title></head><body><h1>Welcome to Site Generator</h1><p>Create and share web pages instantly.</p></body></html>"
}

The generated web page will display the heading "Welcome to Site Generator" and the paragraph "Create and share web pages instantly."
